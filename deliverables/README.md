# Deliverables Package

This folder contains the written project deliverables for the Medical Chatbot assignment. Each file is organized so it can be copied into a single final PDF submission.

## Files

- `01_project_report.md` — formal overview of the project, scope, value, tools, and development approach.
- `02_requirements_documentation.md` — functional and non-functional requirements, plus use cases and process flows.
- `03_design_documentation.md` — architecture, system components, dependencies, data flow, and UI wireframes.
- `04_gantt_chart.md` — task schedule, sequencing, and milestone planning.
- `05_user_documentation.md` — end-user guide for logging in, chatting, and viewing history.
