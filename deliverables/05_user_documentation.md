# User Documentation

## 1. Purpose

This guide explains how to use the Medical Chatbot from an end-user perspective. It is written for students, reviewers, or demo participants who want to understand how the application works in practice.

## 2. Getting Started

1. Open the application in a web browser.
2. If you do not already have an account, open the sign-up page and create one.
3. Verify your email if the system asks for it.
4. Log in using your username and password.
5. Review and accept the privacy notice shown on the chatbot page.
6. You will then be taken to the chatbot home page.

## 3. Using the Chatbot

1. Type a medical or health-related question into the input box.
2. Click **Send** or press Enter if supported.
3. Wait for the chatbot response to appear in the conversation area.
4. Continue asking follow-up questions as needed.

### Good Question Examples

- What are common symptoms of flu?
- What should I do if I have a mild fever?
- Where can I find a nearby clinic?

### Not Recommended

- Emergency situations that require immediate medical assistance
- Requests unrelated to healthcare
- Requests for diagnosis without professional review

## 4. Viewing Chat History

- Open the history page to see your previous conversations.
- Each history record shows the time, the question asked, and the response received.
- Administrators may also view the combined history of all users on the admin history page.

## 5. Data Controls

- Use the **Privacy** link to read how the application handles your chat data.
- Use **Export CSV** on the history page if you want a copy of your stored conversations.
- Use **Delete my history** on the history page if you want to remove your chat records.
- The system masks obvious personal details in stored chat messages to reduce unnecessary exposure.

## 6. Important Pages

- `/signup/` — create a new account
- `/login/` — log in to the application
- `/password-reset/` — request a password reset
- `/` — main chatbot page
- `/history/` — personal chat history
- `/history/all/` — administrator chat history

## 7. Troubleshooting

- If the page looks plain or unstyled, refresh the browser and clear the cache.
- If login fails repeatedly, wait and try again after the lockout period.
- If password reset email does not arrive, check spam/junk and confirm the email settings.
- If chatbot access is blocked, confirm that your email is verified and your account role allows access.

## 8. Safety Note

The chatbot is intended for educational and informational use only. It should not be used for emergency medical decisions. In urgent situations, contact emergency services or a qualified healthcare professional immediately.
